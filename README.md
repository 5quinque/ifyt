# IFYT - Images for your terminal

Print images in any terminal supporting colours


### Building

```bash
make
```

### Running

```bash
./bin/ifyt [-i image path] [-t]
```

### Cleaning

```bash
make clean
```

### Installing

```bash
make install
```
